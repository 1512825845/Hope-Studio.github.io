## VSCode

VSCode 是一个代码编辑器，上手门槛极低(它比 Word 都简单)。

熟练的掌握它的快捷键可以有效提升开发效率，所以可以在后期酌情进行记忆与使用。

## Git

Git 是一个版本管理与多人协作工具，是团队协作项目的必备软件
